# PMPi-PCB
<!-- DESCRIPTION START -->
This is the PCB repository for the PMPi Project, a wireless PM2813 GPIB Controller using a Raspberry Pi Pico W.

Please refer to the [parent repository](https://github.com/LeHuman/PMPi)
<!-- DESCRIPTION END -->

## Status

**`Work In progress`**
